    About

    I am a Founder and Director at The Data Value Factory, a startup helping data users obtain insights from data, and a Research Fellow at the University of Manchester, School of Computer Science, Information Management Group, working in the VADA project. Before that, I was a Senior Researcher with the MCWT/NTUA Research Group and a technical manager in the Hellenic Academic Ebooks project, coordinated by the Hellenic Academic Libraries Link. Until May 2013, I was with the National Documentation Centre, part of the National Hellenic Research Foundation, and until September 2011 with the Autonomic and Grid Computing group in Athens Information Technology as a postdoc. My PhD (2010) and BSc (2004) are from the National Technical University of Athens, School of Electrical and Computer Engineering. I have participated in numerous nationally and internationally-funded research projects and worked in various positions in the software development cycle. I am a member of the Technical Chamber of Greece and a Senior Member of the IEEE. My research has resulted so far in the following publications. If you would like a copy of a paper for which the full text is not provided here, please send me an email.

    Software

    1. Data Preparer is a self-service, hands-off, data preparation/data wrangling platform. No programming or hand-crafting of rules is involved: Users can describe what they need, by providing data sources, a target structure, quality priorities and, optionally, example data. Data Preparer then explores how the data sources relate to each other and the target, and populates the target from the sources. Feedback can then be provided on the automatically produced result or priorities can be revised, and Data Preparer will derive a new result. A free trial version can be used for business evaluation purposes.
    2. The R2RML Parser is a tool that can export relational database contents as RDF graphs, based on an R2RML mapping document.
    3. Biblio-Transformation-Engine is a java framework for filtering and modifying records from various sources, introduced here.
    4. The cerif-jpa-persistence is a java library for accessing CERIF databases with the Java Persistence API.
    5. Expowave is a theoretical framewrok that simulates an RFID reader network that is scheduled using Expowave, an algorithm introduced and analyzed here. You can use RandomGraphGenerator.java to produce random graphs that will have a probability d of being connected to each other. The randomly-generated included graph names denote the number of nodes and d. In order to perform simulation experiments, you have to tweak Simulation.java according to your needs. In short, you can obtain simulation measurements about a node, about the whole graph, or mean measurements about the whole graph as σ (RFID tag birth probability) increases, for each iteration.
    6. The VisAVis Protégé plugin for mapping ontologies to databases was the diploma thesis of Dimitris Spanos and Michalis Chalas. The work was presented here. It uses the Protégé libraries to graphically represent an ontology, a database (MySQL or PostgreSQL) and the mappings between them. The plugin allows queries to be asked to the ontology and returns results from the database. To see VisAVis in action, you have to create a VisAVis folder in Protégé's plugins folder and copy the jar file along with its dependencies into VisAVis folder. Next, create a new Protégé Project and select Project -> Configure -> VisAVisTab. The plugin is tested in Protégé 3.0, 3.1, 3.1.1 and 3.2beta. To compile the source code, you have to extract the code in the plugins folder of Protégé and add to Protégé's MANIFEST.MF the two lines included in VisAVis' MANIFEST.MF.
    7. This java implementation of the board game Score4 (see wikipedia entry) was accomplished in collaboration with Panagiotis Bouros during university. You can play either with another person or with the computer. You can download the source code and the executable jar file.
